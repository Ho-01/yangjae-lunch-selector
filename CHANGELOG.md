# 변경 기록

사용자에게 보이는 주요 변경을 기록한다. 형식은
[Keep a Changelog](https://keepachangelog.com/ko/1.1.0/), 버전은
[유의적 버전](https://semver.org/lang/ko/)을 따른다.

## [미출시]

### 추가

- Google Places의 공식 식음료 기본 타입 166개를 앱의 점심 카테고리 10개로
  변환하는 분류 체계와 누락 검증 테스트
- 세부 타입이 없거나 앞으로 추가될 알 수 없는 타입을 `기타`로 안전하게 처리
- 내 주변 식당의 복수 음식 카테고리 필터와 카테고리별 결과 수
- 음식 카테고리와 독립된 날씨 프로필 테이블 및 기존 설정 자동 이관

### 변경

- 버튼, 텍스트 입력, 선택 상자, 체크박스와 스위치를 shadcn/ui 공통 컴포넌트로 통일
- 기존 식사가챠 색상과 조작 크기를 shadcn/ui 테마와 공통 상태 스타일에 맞게 정리
- 위치·주변 식당 안내에서 API·캐시·세션 같은 개발 용어를 빼고 행동과 결과 중심으로 개선
- 기존 녹색·파란색 전용 디자인을 제거하고 shadcn/ui 기반 Stone + Orange
  의미 색상 체계로 통일
- 과거 디자인 체크리스트와 HTML 참고 시안을 제거
- 화면과 에이전트가 함께 따르는 한글 디자인 규칙 문서 추가
- 내 주변 식당 응답에 Google의 기본 타입과 전체 타입 목록을 포함
- 기존 메뉴 타입을 사용자 화면에서 `날씨 성향`으로 명확하게 표현
- 내 주변 식당은 음식 카테고리가 확률에 영향을 주지 않도록 중립 날씨 프로필 사용
- 모바일 전역 글자 크기, 줄 간격, 버튼 높이, 카드 여백과 긴 이름 줄바꿈 개선
- 양재역 주변의 `성향` 배지와 내 주변의 `음식` 배지를 색상·라벨·아이콘으로 구분
- 내 주변 음식 카테고리 10개에 서로 다른 SVG 아이콘 적용
- 첫 진입 기본 화면을 `양재역 주변`에서 `같이 고르기`로 변경
- 초기 메뉴 로딩에 진행 표시를 추가하고 장소 링크·사진·제외 목록은 화면이 열린 뒤
  이어서 불러오도록 개선
- 내 주변 모드에서 사용하지 않는 날씨 재조회와 방 생성 후 불필요한 목록 대기를 제거
- 진행 중인 보조 요청이 다른 모드의 버튼까지 잠그지 않도록 상호작용 범위 축소

## [0.7.0] - 2026-07-24

### 추가

- 룰렛이 세그먼트 경계를 지날 때 속도에 맞춰 재생되는 틱 효과음
- 당첨 시 짧은 완료음과 색종이 폭죽 효과
- 모션 감소 설정에서는 폭죽을 표시하지 않는 접근성 처리

## [0.6.0] - 2026-07-24

### 추가

- 당첨 항목 경계 안쪽의 무작위 지점에 정지하는 룰렛
- 양쪽 4% 안전 여백을 제외한 당첨 항목 내부 전체의 균등 무작위 정지
- 마지막 느린 이동이 더 오래 보이는 5.4초 감속 곡선
- 포인터는 항상 고정하고 바퀴만 한 방향으로 감속하는 단순한 연출
- 점심방 구성원이 같은 정지 지점을 재현하는 안정 난수
- 모션 감소 환경의 짧고 안정적인 대체 애니메이션
- 작은 세그먼트의 최종 정지 안전성을 확인하는 단위 테스트

## [0.5.0] - 2026-07-24

### 추가

- 방장이 다른 구성원에게 권한을 넘기는 기능
- 현재 사용자와 방장을 명확히 구분하는 구성원 표시
- 닉네임·전송 시각·내 메시지 구분을 지원하는 실시간 방 채팅
- 토큰 검증 채팅 조회·전송 RPC와 최근 100개 메시지 보관
- 메시지 300자 제한과 과도한 연속 전송 방지

## [0.4.0] - 2026-07-24

### 추가

- 로그인 없이 기존 구성원으로 점심방에 다시 들어가는 기능
- 상태·인원·최근 활동·만료 시간과 재입장 버튼을 제공하는 최근 참여 방 목록
- 기존 단일 세션을 최근 방 최대 10개 형식으로 자동 변환
- 해시된 구성원 토큰을 검증하는 재입장 RPC

### 변경

- 화면과 훅에서 Supabase 구현을 분리하는 백엔드 포트·어댑터 구조 도입
- 향후 Spring·PostgreSQL 이전을 위한 계약과 데이터 이관 원칙 문서화

## [0.3.0] - 2026-07-24

### 추가

- 일반·내 주변·점심방 결과의 이미지 카드 공유
- SVG 아이콘과 함께 표시되는 무작위 결과 멘트
- 브라우저에 저장되는 최근 결과 최대 10개와 전체 삭제
- 최근 3개 메뉴의 가중치를 선택적으로 낮추는 기능
- 날씨 가중치 ON/OFF와 오른쪽 제외 메뉴 목록 검색

### 변경

- 공개 제품명을 앱·메타데이터·공유·문서에서 `식사가챠`로 통일
- 이모지 문자를 `react-icons` SVG 아이콘 키 방식으로 변경

## [0.2.0] - 2026-07-24

### 추가

- UX 검증표와 키보드 접근성 기준
- 데스크톱·320px·390px 브라우저 E2E 테스트
- 모바일 보조 패널 접기
- 식당 검색과 후보 이름 직접 입력의 명확한 분리
- 로딩·빈 상태·오류·재시도 표현 통일
- Supabase migration 자동 배포와 상태 검증

### 변경

- 삭제·후보 제거·투표 마감에 확인 절차 추가
- 위치와 네트워크 실패의 복구 안내 개선
- Supabase 수동 적용 기록을 복구하고 일회성 repair 옵션 제거

## [0.1.0] - 2026-07-24

### 추가

- 날씨 가중치와 오늘 제외 목록을 지원하는 팀 메뉴 룰렛
- 메뉴·메뉴 종류와 날씨 가중치 관리
- Google Places 연결, 별점, 사진과 Supabase Storage 사진 캐시
- 브라우저 위치와 30분 캐시를 사용하는 내 주변 모드
- 후보·좋아요·거부권·준비 상태·동기화 룰렛을 지원하는 점심방
- 활동 기록, 재촉 알림, 닉네임 수정과 결과 공유
- 모바일 반응형 화면과 Supabase 감사 기록

### 수정

- Linux/Vercel 빌드 재현성과 Supabase migration 호환성
- 점심방 모드 격리, 상태 표시와 동기화된 룰렛 유지
